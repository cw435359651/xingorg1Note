## 提示 格式

::: tip 提示
This is a tip
:::

::: warning 警告
This is a warning
:::

::: danger 危险
This is a dangerous warning
:::

## 代码中的行高亮

语言后面使用{数字} 例如 js{4} 表示第四行高亮

```js{2}
var a = 1;
var b = 2;
var c = 3;
```
