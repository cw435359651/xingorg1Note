## 快速起步

安装：

```bash
npm i
# OR
cnpm i
```

运行：

```bash
npm run dev
# OR
npm run serve
```

打包：

```bash
npm run build
```

自动发布：

```bash
npm run push "git commit提交文案，不能有空格"
```
